# 🧑‍💻 Learn to Code — A Month-Long Course for Beginners

> Built for non-developers who want to learn modern coding — including AI tools — in their evenings and free time. No prior experience needed.

---

## 📅 Course Structure

**4 weeks · 4–5 sessions per week · 30–60 min each**

| Week | Theme | Focus Areas |
|------|-------|-------------|
| [Week 1](./week-1/) | Foundations | Terminal, Git, GitHub, HTML & CSS |
| [Week 2](./week-2/) | Programming Basics | Python fundamentals |
| [Week 3](./week-3/) | Building Real Things | APIs, automation, databases |
| [Week 4](./week-4/) | AI/ML Era Coding | GitHub Copilot, LLMs, deploying apps |

> 🗓️ **New week's lessons drop every Sunday at 10am** — check back here each week!

---

## 🛠️ What You'll Need

- A computer (Mac, Windows, or Linux)
- A GitHub account (you already have this!)
- VS Code — free code editor: https://code.visualstudio.com/
- Curiosity and ~30 mins a day

---

## 🎯 What You'll Build By the End

- A personal webpage (HTML & CSS)
- A Python script that automates a boring task
- A script that pulls live data from the internet
- An app that uses an AI API (like OpenAI or Claude)
- A project deployed online that real people can visit

---

## 📖 How to Use This Course

1. Work through lessons in order — each one builds on the last
2. Every lesson has a concept explanation, hands-on exercise, and optional challenge
3. Commit your work to this repo as you go — that is real-world practice
4. Google and AI assistants are encouraged — that is how professional developers actually work
5. Do not worry about understanding everything perfectly. Momentum beats perfection.

---

## 🗓️ Week-by-Week Overview

### Week 1 — Foundations
Get comfortable with the tools every developer uses daily.
- Lesson 1: The Terminal — talking to your computer like a developer
- Lesson 2: Git — saving your work like a pro
- Lesson 3: GitHub — sharing code with the world
- Lesson 4: HTML — building the skeleton of a webpage
- Lesson 5: CSS — making your webpage look good

### Week 2 — Programming Basics (unlocks Sunday Week 2)
Learn to think like a programmer using Python.
- Lesson 1: Variables and Data Types
- Lesson 2: Functions — reusable blocks of logic
- Lesson 3: Loops and Conditionals
- Lesson 4: Lists, Dictionaries and Data Structures
- Lesson 5: Your first real Python script

### Week 3 — Building Real Things (unlocks Sunday Week 3)
Connect your code to the real world.
- Lesson 1: What is an API? (and why developers love them)
- Lesson 2: Fetching live data from the internet
- Lesson 3: Reading and writing files, working with CSVs
- Lesson 4: Intro to databases — storing data
- Lesson 5: Automating a real task

### Week 4 — AI/ML Era Coding (unlocks Sunday Week 4)
Code in the age of AI — the skills that matter most right now.
- Lesson 1: GitHub Copilot and AI code assistants
- Lesson 2: Calling AI APIs (OpenAI, Claude, Gemini)
- Lesson 3: Prompt engineering for developers
- Lesson 4: What is ML? Training vs inference, real-world use cases
- Lesson 5: Deploy your project — put it on the internet

---

## 💡 Tips from Developers

- Stack Overflow and Google are your friends — every developer uses them constantly
- Read error messages carefully — they tell you exactly what went wrong
- Break big problems into small steps — this is literally what coding is
- AI assistants like GitHub Copilot or Claude can explain code, fix bugs, and teach you
- The best way to learn is to build something you actually want
